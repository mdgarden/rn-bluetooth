## 실행방법

### 안드로이드 에뮬레이터 실행 (Mac)

- Android Studio 실행
- projects에서 device manager 실행
- 플레이 버튼 눌러서 에뮬레이서 실행
- vs code 터미널에서 `npx expo run:android` 실행
